\` a * b = c   => a = c / b => b = c / a\`


\` a / b = c  => a = b * c\`

\` 1 / 2 = 0.5 =>  1/0.5 = 2 \`

\` 12 / 3 = 4 => 12 / 4 = 3 \`

\` 18 / 3 = 6 => 18 / 6 = 3 \`

\` 面积 = 长度 *  宽度 \`

\` 长度 = (面积) / (宽度) \`





[asciimath](http://asciimath.org) 在页面中加入下面这段代码

```html

```
